Eyebrow — small uppercase kicker label set in the typewriter "Special Elite" face, used above every section title.

```jsx
<Eyebrow>Asesoría migratoria · Vigo, Galicia</Eyebrow>
<Eyebrow tone="dark">Qué tramitamos</Eyebrow>
<Eyebrow tone="light">Cómo trabajamos</Eyebrow>
```

Use `tone="light"` only on the dark forest-green methodology section background.
