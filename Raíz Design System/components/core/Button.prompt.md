Button — the site's call-to-action control. Three variants: solid forest `primary`, outlined `secondary`, and green `whatsapp` (used for the main "Escríbenos por WhatsApp" CTA).

```jsx
<Button variant="primary" size="large" href="#contacto">Evaluar mi caso por WhatsApp</Button>
<Button variant="secondary" href="blog.html">Ver todos los artículos</Button>
<Button variant="whatsapp" href="https://wa.me/...">💬 Escríbenos por WhatsApp</Button>
```

Notable variants/props: `size="large"` is used for the hero's main CTA; `fullWidth` is used for form submit buttons.
