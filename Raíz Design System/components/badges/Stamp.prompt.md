Stamp — the rotated, bordered "ink stamp" label used to mark a problem as REJECTED or a Raíz solution as APPROVED.

```jsx
<Stamp tone="rejected">RECHAZADO</Stamp>
<Stamp tone="approved">APROBADO</Stamp>
```

Pair with the `stamp-texture` SVG filter (see assets/brand/texture-filters.svg) for the authentic ink-on-paper distortion seen on the live site.
